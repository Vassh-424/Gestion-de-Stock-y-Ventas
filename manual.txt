Bienvenido al sistema de Gestión de Stock y Ventas V1. Este programa permite gestionar productos, registrar ventas, consultar el historial de ventas, y realizar modificaciones en el inventario de forma eficiente. A continuación, se detallan las funcionalidades principales y su uso.



Pantalla Principal

Al iniciar el programa, se mostrará la pantalla principal, donde se encuentran las siguientes opciones:



Stock: Permite gestionar el inventario de productos (agregar, buscar, modificar, y eliminar productos).

Nueva Venta: Accede al módulo de registro de ventas.

Historial de Ventas: Permite consultar las ventas realizadas.

Manual: Abre este manual de usuario para orientación sobre el uso del programa.

Salir: Cierra la aplicación de manera segura.
Funcionalidades de Gestión de Stock

Al seleccionar la opción Stock, se abrirá una nueva ventana titulada "Gestión de Stock". En esta ventana, tienes las siguientes opciones:


Agregar Producto:
Permite añadir un nuevo producto al inventario.


Buscar Producto:
Abre una ventana para buscar productos específicos y consultar su disponibilidad en el inventario.


Modificar Producto:
Permite editar los detalles de un producto existente (por ejemplo, cambiar el precio o actualizar la cantidad disponible).


Eliminar Producto:
Muestra una ventana para eliminar un producto del inventario.


Volver:
Regresa a la pantalla principal sin realizar ninguna modificación.

Cada opción dentro del menú de Stock se activa mediante un botón con texto claro y tamaño grande para facilitar su uso.
